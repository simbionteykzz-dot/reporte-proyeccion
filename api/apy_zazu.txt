========================================
  ZAZU EXPRESS — integración panel reportes
========================================

El panel «Envíos diarios (Zazu)» llama a tu proyecto Flask:
  GET /api/zazu/envios-diarios?tab=entregados|anulados&limit=1000

El backend consulta Supabase PostgREST (misma consulta que documentaste):
  - Tabla: tb_envios_diarios_lina
  - select: *, envío y motorizado embebidos
  - Entregados: estado_pedido=Entregado y verificacion=true
  - Anulados: estado_pedido=Anulado

Variables de entorno en el SERVIDOR (no en el navegador):
  ZAZU_SUPABASE_URL=https://<ref>.supabase.co
  ZAZU_SUPABASE_ANON_KEY=<JWT anon>
  # Opcional, si RLS exige JWT de sesión:
  # ZAZU_SUPABASE_JWT=<token>

Salud del despliegue:
  GET /api/health  → deployment.zazu_supabase_configured
